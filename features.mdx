## API traffic monitoring

- Monitor API traffic in near real-time, per endpoint, consumer, and across multiple environments
- View error rates, including both client and server errors
- View different percentiles and a histogram of response times of endpoints
- View a breakdown of status codes returned by endpoints
- View details about validation errors users are encountering (FastAPI and Django Ninja only)
- List all endpoints of your application (even those that haven't been used)
- Filter to different time periods and individual consumers
- Data retention of up to one year

## API key management

- Easily set up API key authentication in your application with our client libraries
- Create and manage API keys for your application
- Define scopes and assign them to API keys for fine-grained access control
- Select which environments API keys are valid in
- Set expiration dates on API keys
- Rotate API keys with a grace period to avoid service interruptions
- Disable API keys to immediately revoke access (and re-enable them later if needed)
- Track the usage of API keys over time
- Import existing API keys

## General

- Support for FastAPI, Starlette, Flask, Django Ninja and Django REST Framework (more soon)
- Add and manage applications and their environments
- View number of active instances of each application per environment
- Create and switch between different teams
- Invite team members with owner, admin or read-only permissions

## Roadmap
