title: "Quickstart"
description: "Get started in under 5 minutes."
---

Follow the 3 simple steps below to start monitoring your API traffic, errors and response times with Apitally.

<Steps>
  <Step icon="user" iconType="regular" title="Log in to our dashboard">
    [Sign up](https://app.apitally.io/?signup) to create an account, if you
    don't have one yet. It's free and only takes a few seconds. We support
    logging in with your GitHub account too.
  </Step>
  <Step icon="plus" iconType="regular" title="Create app">
    Create a new app in the dashboard. You'll only need to provide a name and
    select the framework your application is using.
  </Step>
  <Step
    icon="code-pull-request"
    iconType="regular"
    title="Add middleware to your application code"
  >
    Simply follow the instructions on the dashboard to add the middleware to
    your application code. Then deploy your application when you're finished.
  </Step>
</Steps>

For detailed setup instruction, including how to identify consumers or set up API key authentication, select your framework from the options below.
