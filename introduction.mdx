## Quickstart

Learn how to start monitoring your REST API in less than 5 minutes.

<Steps>
  <Step icon="user-plus" iconType="regular" title="Log in to the Apitally dashboard">
    [Sign up](https://app.apitally.io/?signup) to create an account, if you
    don't have one yet. It only takes a few seconds.
  </Step>
  <Step icon="plus" iconType="regular" title="Create your app">
    Click the *Create app* button on the dashboard. You'll only need to provide
    a name and select the framework your application is using.
  </Step>
  <Step
    icon="code-pull-request"
    iconType="regular"
    title="Add middleware to your application"
  >
    Follow the instructions on the dashboard to add the Apitally middleware to
    your application. It's just a few lines to copy & paste. Then deploy
    your application.
  </Step>
</Steps>

For more detailed instructions, including how to identify API consumers, select your framework from the options below.
