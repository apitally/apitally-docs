Apitally is a simple and affordable API monitoring and API key management solution with a focus on data privacy.
It is easy to set up and use for new and existing API projects using Python or Node.js.

![Screenshots](https://assets.apitally.io/screenshots/overview.png)

## Key features

- [API traffic monitoring](/features/api-traffic-monitoring):
Monitor API traffic, errors, and response times on an intuitive dashboard.
Get insights into the usage and performance of each endpoint.
Filter by API consumer to understand different usage patterns.

- [API uptime monitoring & alerting](/features/api-uptime-monitoring):
Monitor API uptime and availability.
Get instant notifications via email if your API goes down.

- [API key management](/features/api-key-management):
Easily secure your endpoints with API key authentication.
Manage API keys, scopes and permissions on our simple dashboard.
Track API key usage and expire or rotate API keys without causing disruptions.

## Why Apitally?

- Never collects or stores sensitive data. Captures only non-sensitive metadata.
- Integrates directly with your applications via middleware and does not require any infrastructure changes. No impact on performance.
- Supports the most popular Python and Node.js web frameworks through open-source client libraries.
- Works for internal APIs in private subnets just as well as public-facing APIs.
