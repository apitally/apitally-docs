By clicking on a row in the [endpoints table](/features/traffic-dashboard#endpoints-table) on the traffic dashboard, you can access detailed insights into the usage and performance of that specific API endpoint.

The endpoint insights are presented in a tabbed interface with the sections described below.

## Info

<Card>
  <div style={{ margin: "-1.25rem -1.5rem" }}>
    <img src="https://assets.apitally.io/docs/endpoint-insights-info.webp" alt="Endpoint insights - Info tab" />
  </div>
</Card>

The *Info* tab presents metadata about the endpoint, such as the summary and description, if provided by your application (e.g. via the OpenAPI schema).

## Data transferred

<Card>
  <div style={{ margin: "-1.25rem -1.5rem" }}>
    <img src="https://assets.apitally.io/docs/endpoint-insights-data-transferred.webp" alt="Endpoint insights - Data transferred tab" />
  </div>
</Card>

Apitally records the payload size of each API request and response based on the `Content-Length` headers.
It provides insights into the distribution of payload sizes for specific endpoints as well as the overall amount of data transferred.
